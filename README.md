# Expense Tracker (Python CLI App)

A simple yet powerful Expense Tracker built with Python that helps users manage daily expenses, monitor spending habits, and export records for analysis.

## Features

- Add expenses
- View all expenses
- Edit expenses
- Delete expenses
- Search expenses by category
- Monthly spending summary
- Category-wise analytics
- Budget alert system
- Export data to CSV
- Automatic date and time logging

## Tech Stack

- Python
- JSON (data storage)
- CSV
- OOP
- File Handling
- Data Analytics Logic

## Learning Outcomes

This project helped strengthen:

- Python OOP
- Functions & Modular Programming
- JSON handling
- File handling
- Data analytics logic
- Error handling
